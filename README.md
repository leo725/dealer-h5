# 参数说明：
店铺分享： dId - 车商id, sId - 员工id, shareId - 营销工具分享id
线下店分享： mId - 线下店id, sId - 员工id, city - 线下店所属城市的cityCode, shareId - 营销工具分享id
专题分享（车商、线下店所需参数同上）: ids - 所选车源id集合(1,2,3,4), bannerId - banner图片的id
